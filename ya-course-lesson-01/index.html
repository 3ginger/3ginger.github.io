Подготовим окружение для первого проекта

  • Установить Node.js и npm
  • Перейти в папку с проектом (терминал)
  • Инициализировать пакетный менеджер: npm init
  • Установить пакеты: npm i d3 lodash normalize.css --save
  • Завести структуру проекта

Структура проекта

.
├── data
├── images
├── index.html
├── main.css
├── main.js
└── package.json

Установленные пакеты

  • d3 – пакет для визуализации данных
  • lodash – пакет для манипуляции данными
  • normalize.css – пакет для приведения css к единому стилю
